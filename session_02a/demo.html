How the dropdown works:

  1. HTML Structure: Main menu (ul) with nested submenu (ul) inside a list item
  2. CSS Position: Parent li has "position: relative", dropdown has "position: absolute"
  3. Hidden by default: Dropdown has "display: none"
  4. Show on hover: ":hover" selector changes to "display: block"
  5. Positioning: "top: 100%" places dropdown right below the parent menu item

Key CSS Properties:

  • position: relative on parent creates positioning context
  • position: absolute on dropdown removes it from normal flow
  • display: none/block controls visibility
  • :hover pseudo-class triggers the dropdown
